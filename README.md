## 📖 Deskripsi Proyek
**Sehat Walafiat** adalah rancangan aplikasi berbasis *mobile* untuk pemantauan kesehatan harian yang dirancang khusus dengan prinsip **"Zero Learning Curve"**. Aplikasi ini menyasar pengguna dari kalangan **Lansia** yang awam teknologi. 

Sistem beroperasi secara otomatis dengan menarik data biometrik dari *Smartwatch* via Bluetooth Low Energy (BLE), dan interaksi utama dikendalikan menggunakan **Voice Command** (Perintah Suara). Aplikasi ini juga dilengkapi dengan ekosistem **Family Link** untuk pemantauan keluarga dari jarak jauh serta sistem Peringatan Darurat Medis cerdas.
